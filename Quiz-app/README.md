#  React Quiz App 

A Simple React Quiz App which contains the 30  MCQ’s which have 4 options in each.


## 📝 Usage

### Clone this Repository
```sh
git clone https://github.com/PuviGS/WebDevProjects.git
```
### Navigate to Quiz-app
 ```sh
 cd Quiz-app
```


### Installation

```sh
$ npm install
```

### ✔️ Run

```sh
$ npm start
```
